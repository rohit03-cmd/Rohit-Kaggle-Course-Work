## ‚öôÔ∏è Getting Started

To run the notebooks locally, please follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/](https://github.com/)[YourGitHubUsername]/[YourName]-Kaggle-Learning-Path.git
    ```
2.  **Create Environment (Recommended):**
    ```bash
    conda create -n kaggle-env python=3.9
    conda activate kaggle-env
    ```
3.  **Install Dependencies:**
    ```bash
    pip install pandas numpy scikit-learn matplotlib seaborn jupyter
    ```
4.  **Launch Jupyter:**
    ```bash
    jupyter notebook
    ```
    Navigate to the desired course folder to view the solutions.
